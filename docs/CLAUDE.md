# docs/

## Purpose

Challenge brief and reference data from the Syngenta Mars Crop Knowledge Base (extracted via MCP server).

## Key Files

- `CHALLENGE.MD` — START Hack challenge brief and requirements
- `mcp-data/` — 7 reference documents extracted from the Syngenta KB:
  - `01_MARS_ENVIRONMENT_EXTENDED.MD` — Mars surface conditions, radiation, dust storms
  - `02_CONTROLLED_ENVIRONMENT_AGRICULTURE_PRINCIPLES.MD` — CEA fundamentals
  - `03_CROP_PROFILES_EXTENDED.MD` — Crop growth parameters, yields, requirements
  - `04_PLANT_STRESS_AND_RESPONSE_GUIDE.MD` — Stress detection and mitigation
  - `05_HUMAN_NUTRITIONAL_STRATEGY.MD` — Crew dietary requirements for 450 days
  - `06_GREENHOUSE_OPERATIONAL_SCENARIOS.MD` — Crisis scenarios and responses
  - `07_FROM_MARS_TO_EARTH_INNOVATION_IMPACT.MD` — Earth applications

## How to Use

- Agent decision-making should reference crop profiles and stress guide
- Simulation scenarios come from `06_GREENHOUSE_OPERATIONAL_SCENARIOS.MD`
- Nutritional targets from `05_HUMAN_NUTRITIONAL_STRATEGY.MD`
